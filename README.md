#### 当前实现的功能有

- 登录
- 搜索
- 播放（自动过滤无版权以及付费歌曲）
- 歌单列表

#### 当前项目所用库

- react(react hooks)
- react-router
- redux
- axios
- tiny-swiper
- scss/styled-components
